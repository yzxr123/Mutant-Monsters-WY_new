# 究极突变生物 网易版

这是一个用于网易我的世界的突变生物组件
项目目标是在网易我的世界中实现大型突变生物  Boss 技能  特殊动画  特殊掉落  多阶段战斗机制

## 安装说明

1. 下载本仓库 Release 中的组件包
2. 导入网易我的世界开发环境
3. 启用行为包和资源包
4. 进入世界测试突变生物实体和技能效果

## 开发说明

本项目主要面向网易我的世界组件开发
脚本部分使用 Python
资源结构遵循网易我的世界基岩版组件结构
